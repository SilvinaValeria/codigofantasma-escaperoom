Esta web está pensada para formar parte de un acertijo en una sala de escape de la Escuela Técnica N°26 "Confederación Suiza".
Consta de un formulario, con dos inputs para ingresar el usuario y contraseña y una vez descubiertas las credenciales, se mostrará un ALERT con el código necesario para continuar con el siguiente acertijo de la sala de escape.
